How to present code

When including code snippets use preformatted style as below. In general try to avoid using specific fonts.

public void createPartControl(Composite parent) { /* [1] */
	label = new Label(parent, 0); 
	label.setText("Hello World"); / * [2] */
	getViewSite().getPage().addSelectionListener(this);
}

Consider annotating your code, if necessary using comments appropriate for the language.

In the preceding example, [1] is the signature of the method and [2] sets text of the label.

Source Code

Always include a link to your zip file containing the source code. You may want to include a section titled Source Code (as we have done here) with the text shown below. Remember to ensure the zip file does not include eclipse/plugins in the path of files; it should only include only your plug-in directory name. Users will assume they can unzip the plug-in directly into their plugins directory. Here is some sample text:

To run the example or view the source code for this article unzip your_filename.zip into your plugins/ subdirectory.
